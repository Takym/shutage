## 命名規則 <a id="naming_convention"></a>
* `_posts`ディレクトリ内のファイルは以下の様に命名する。
* 2/5chまとめのSS作品の場合：
	* `yyyy-mm-dd-ssxxxx` (`yyyy`は年、`mm`は月、`dd`は日を表す。四桁の数字の前に`ss`を付ける。)
* このリポジトリのオリジナル作品の場合：
	* `yyyy-mm-dd-myssxxxx` (`yyyy`は年、`mm`は月、`dd`は日を表す。四桁の数字の前に`myss`を付ける。)
	* `自作`タグを付与する。
